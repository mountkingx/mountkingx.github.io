Summary

I am experienced in web application and web services development. My developer journey started from the J2EE web course at CMU. MySQL and Redis are the primary databases I used, though for some projects MongoDB was also chosen. Right now I am exploring the Spring Cloud.
I also have a great passion for Cloud Computing and Big Data. I have accomplished multiple challenging projects, from Java code-driven auto scaling of web servers to provisioning Hadoop EMR clusters in Amazon Web Services for ETL of large data sets, from creating Lambda functions for event-triggered service to optimizing DB schema for handling high concurrent requests. Cloud is exciting, and more to learn and apply.
